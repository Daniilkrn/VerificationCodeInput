# VerificationCodeInput

Демо-проект с авторизацией по email и вводом кода подтверждения.

## Что сделано

- Реализован пользовательский сценарий входа: `email -> код подтверждения -> экран успеха`.
- Добавлена отправка 6-значного кода через EmailJS.
- Реализована проверка кода на клиенте с ограничением времени действия (10 минут).
- Сделан удобный компонент ввода кода:
  - автопереход между ячейками;
  - вставка полного кода из буфера обмена;
  - навигация стрелками;
  - корректная обработка Backspace/Delete;
  - отображение ошибки и успешного состояния.
- Добавлен переход на ввод другого email (сброс текущей сессии подтверждения).
- Реализован анимированный WebGL-фон для экрана авторизации.

## Технологии

- React 18
- TypeScript
- Vite
- SCSS Modules
- EmailJS (`@emailjs/browser`)
- Three.js
